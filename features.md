# ✨ 주요 기능 (Feature Showcase)

Docling Translate가 제공하는 강력한 기능들을 소개합니다.

## 1. 다양한 포맷 지원 (Multi-Format Support)
PDF뿐만 아니라 DOCX, PPTX, HTML, 이미지 파일까지 지원합니다. 어떤 형태의 문서든 Docling Translate 하나면 충분합니다.

- **PDF**: 논문, 기술 문서, 매뉴얼
- **DOCX/PPTX**: 업무용 문서, 프레젠테이션 자료
- **Image**: 스캔된 문서, 포스터, 인포그래픽

## 2. 로컬 LLM 완벽 지원 (Local LLM Support)
클라우드 API 비용이 걱정되시나요? 보안이 중요한가요?
Qwen, LFM2, Yanolja 등 고성능 로컬 LLM을 사용하여 **오프라인** 환경에서도 무료로 번역할 수 있습니다.

## 3. 인터랙티브 HTML 뷰어 (Interactive Viewer)
번역된 결과물은 단순한 텍스트 파일이 아닙니다. 웹 브라우저에서 열 수 있는 인터랙티브 HTML 파일로 제공됩니다.
- **Split View**: 원문과 번역문을 좌우로 비교
- **Click to Expand**: 번역문을 클릭하면 원문이 펼쳐짐

## 4. 고성능 병렬 처리 (High Performance)
수백 페이지의 문서도 문제없습니다. 멀티스레딩 기술을 적용하여 문장 단위로 병렬 번역을 수행하므로 처리 속도가 매우 빠릅니다.

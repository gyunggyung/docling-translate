# Issue #52: 불필요한 기능 제거

## 목표

로컬 실행 환경에서 불필요한 기능(파일 다운로드 등)을 제거하여 코드를 경량화하고 사용자 경험을 단순화합니다.

## 배경

현재 `app.py`는 웹 배포를 가정하여 변환된 파일(ZIP, HTML)을 다운로드하는 기능을 제공합니다. 하지만 로컬 환경에서는 사용자가 출력 폴더(`output/`)에 직접 접근할 수 있으므로, 이 기능은 중복이며 불필요합니다. 또한, 다운로드 기능 제거 시 `create_zip`과 같은 유틸리티 함수도 더 이상 필요하지 않게 됩니다.

## 제안 변경 사항

### UI (Streamlit)

#### [MODIFY] [app.py](file:///c:/github/docling-translate/app.py)

**변경 내용**:
- **ZIP 다운로드 버튼 제거**: `st.download_button` (ZIP) 제거.
- **HTML 다운로드 버튼 제거**: `st.download_button` (HTML) 제거.
- **폴더 열기 버튼 유지**: 로컬 환경에서 유용한 `os.startfile` 기능은 유지.
- **관련 코드 정리**: `create_zip` 호출 제거.

### 유틸리티

#### [MODIFY] [src/utils.py](file:///c:/github/docling-translate/src/utils.py)

**변경 내용**:
- **`create_zip` 함수 제거**: 더 이상 사용되지 않음.
- **`zipfile` 임포트 제거**: 불필요한 라이브러리 의존성 제거.

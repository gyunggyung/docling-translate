# 자주 묻는 질문 (FAQ)

## 일반 (General)

### 이 도구는 무료인가요?
네, Docling Translate는 오픈소스이며 무료로 사용할 수 있습니다. 단, OpenAI나 DeepL 같은 타사 API를 사용하는 경우 해당 서비스 제공자에게 비용을 지불해야 할 수 있습니다. 로컬 LLM(Qwen, LFM2 등)을 사용하면 비용 없이 사용할 수 있습니다.

### 오프라인에서 실행할 수 있나요?
네, 가능합니다. Qwen, LFM2, Yanolja 등 로컬 LLM을 지원하므로 인터넷 연결 없이도 번역 작업을 수행할 수 있습니다.

## 기술 (Technical)

### 어떤 파일 형식을 지원하나요?
PDF, DOCX, PPTX, HTML, Image 등 다양한 포맷을 지원합니다. 모든 포맷은 번역 후 인터랙티브 HTML 뷰어로 변환됩니다.

### 번역 정확도는 어느 정도인가요?
사용하는 모델(예: GPT-5-nano, DeepL)에 따라 다릅니다. 일반적으로 기술 문서에 대해서는 매우 높은 정확도를 보입니다. 레이아웃 보존 기능 덕분에 문맥 파악이 용이합니다.

### 복잡한 레이아웃의 PDF도 잘 되나요?
Docling은 다단 편집이나 표가 포함된 복잡한 레이아웃을 처리하도록 설계되었습니다. 하지만 매우 비표준적인 PDF는 일부 오류가 있을 수 있습니다.

# 유지보수 및 문제 해결 (Maintenance and Troubleshooting)

## 일반적인 문제 (Common Issues)

### 1. "API Key Missing" 오류
**증상**: API 키가 없다는 오류 메시지와 함께 프로그램이 종료됩니다.
**해결**: 루트 디렉토리에 `.env` 파일이 있는지, 그리고 올바른 키(예: `OPENAI_API_KEY=...`)가 입력되었는지 확인하세요.

### 2. PDF 파싱 실패
**증상**: 특정 PDF에서 텍스트를 추출하지 못합니다.
**해결**:
- PDF에 암호가 걸려 있는지 확인하세요.
- 스캔된 PDF(이미지)인 경우 OCR이 필요할 수 있습니다. Docling이 이를 처리하지만, 필요한 의존성이 설치되어 있는지 확인하세요.
- 파일이 손상된 경우 표준 포맷으로 다시 저장해 보세요.

### 3. 번역 품질 문제
**증상**: 번역이 부정확하거나 문맥에 맞지 않습니다.
**해결**:
- 다른 모델을 시도해 보세요 (예: `gpt-3.5-turbo` 대신 `gpt-4o` 사용).
- 원본 텍스트의 서식이 너무 복잡하여 모델이 혼동하는지 확인하세요.

## 유지보수 (Maintenance)

### 의존성 업데이트
주기적으로 다음 명령어를 실행하여 라이브러리를 최신 상태로 유지하세요:
```bash
pip install -r requirements.txt --upgrade
```

### 임시 파일 정리
`temp_uploads/` 또는 `output/` 폴더에 임시 파일이 쌓일 수 있습니다. 더 이상 필요하지 않다면 삭제해도 안전합니다.
